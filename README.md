**💰 Smart Expense Tracker**

A feature-rich and beginner-friendly Smart Expense Tracker Web Application built using HTML, CSS, and JavaScript (No Backend, No Frameworks).
This project helps users manage their daily expenses efficiently with a clean UI and powerful features.

---

**🚀 Features**

**🔐 Authentication**

- Simple login system using localStorage
- Unique username & password validation
- Protected pages (Dashboard, Charts, Reports)
- Logout functionality

---

**🏠 Dashboard**

- Add expenses manually
- Edit & delete expenses
- Optional notes for each expense
- Budget setting with progress bar
- Real-time total, balance, and entry count

---

**📂 CSV Import / Export**

- Upload expenses using CSV file
- Format:
  Name,Amount,Category,Date,Notes
- Prevent duplicate entries while importing
- Download/export expenses as CSV

---

**📊 Advanced Functionalities**

- Sort expenses (Amount & Date)
- Category icons (🍔 ✈️ 🛍️ 💡 etc.)
- Monthly expense summary
- Top spending category detection
- Recent transactions (last 3–5)
- Clear expenses by category
- Duplicate expense prevention

---

**🧠 Smart Insights**

- “You are spending more on Food”
- “Your spending increased this week”

---

**🎨 UI / UX Features**

- Clean dashboard layout with sidebar
- Gradient design with card-style UI
- Dark/Light mode (saved in localStorage)
- Smooth animations
- Highlight high expenses
- Confirmation popups for delete/clear
- Notifications for actions

---

**📈 Charts Page**

- Pie Chart (Category-wise expenses)
- Bar Chart (Date-wise expenses)
- Built using Chart.js CDN

---

**📄 Reports Page**

- Expense summary
- Total and count display
- CSV download option

**▶️ How to Run the Project**

1. Download or clone the repository
2. Open the project folder in VS Code
3. Open login.html
4. Run using Live Server or open in browser

---

**🔑 Login Instructions**

- Create a username (must be unique)
- Password must:
  - Be at least 6 characters
  - Include letters and numbers

---

**⚠️ Notes**

- This project uses localStorage, so data is stored only in your browser
- No backend or database is used
- Works best in modern browsers (Chrome, Edge)

---

**🎯 Project Goal**

To build a real-world, user-friendly expense tracking application with advanced features while keeping the code simple and easy to understand for beginners.
